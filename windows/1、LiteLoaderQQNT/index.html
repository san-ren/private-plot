    2023年，腾讯推出基于electron的QQNT，抛弃尾大不掉的代码和上世纪风格的ui

    虽说跟TIM相比，不能说很像只能说一模一样

    但对于一个TIM用户来说，喜闻乐见

    浏览器般的本质，使其拥有极高的操作性和可玩性，就像火狐

    LiteLoaderQQNT¶

    官网：https://liteloaderqqnt.github.io/

    全局¶

    项目 徽章 说明
    LL-plugin-list-viewer


    		插件列表 查看·安装·更新
    PluginInstaller


    		插件安装器
    lite_tools


    		轻量工具箱
    transitio


    		加载任意用户样式
    Scriptio


    		加载任意渲染层的 JavaScript 片段
    Custom-CSS


    		加载任意CSS
    Kill-Update


    		关闭自动更新弹窗
    QQCleaner


    		缓存清理插件

    消息本身¶

    项目 徽章 说明
    qqMessageBlocker

    		QQ信息消息屏蔽插件，支持按关键字和用户名屏蔽文字、emoji表情、图片表情包、群机器人、超级表情。
    Stick-Emoji

    		自动给消息贴上emoji！(๑˃ᴗ˂)ﻭ
    Remove-Custom-Bubbles

    		干掉QQ在9.9.16.29725版本开始加入的自定义气泡皮肤，并恢复原版极简样式
    QR-Decode

    		聊天中的图片二维码解析
    pURLfy

    		净化 QQNT 中的 URL 链接
    Auto-Format-Message

    		格式化消息，排版要发送的中英混文消息 （人话：加空格）
    Anti-Recall

    		简易防撤回

    消息外围¶

    项目 徽章 说明
    QuickReply

    		快捷回复
    AutoSendMessages

    		定时发送消息
    AutoForward

    		自动转发QQ消息至指定的好友、群聊或群内成员
    Directly-Jump

    		使用自定义浏览器打开链接并跳过拦截页
    MergeMessage

    		消息聚合
    qqMessageSave

    		QQ消息保存
    Quickly-Remove-Conversations

    		基于 LiteLoader 的滚轮（或自定义快捷键）快速删除会话窗口的插件
    HotkeyChangeWindow

    		通过快捷键快速跳转聊天窗口
